What is Reconciliation?

Reconciliation is the process by which React updates 
the UI when state or props change.

In simple words:
When something changes ‚Üí
React must decide:
What changed?
What stayed same?
What to update in the real DOM?
That entire decision-making process is called Reconciliation.

What is Diffing?

Diffing is a part of Reconciliation.
Definition:
Diffing is the algorithm React uses to compare 
the old Virtual DOM with the new Virtual DOM 
to find differences.
React does NOT compare real DOMs.
It compares:
Old Virtual DOM
vs
New Virtual DOM
That comparison process = Diffing.

#### Full Flow of What Happens

Let‚Äôs say you use useState:
setCount(count + 1);

Now internally:
Step 1: State Update
React marks component for re-render.

Step 2: Re-render
The component function runs again.

Step 3: New Virtual DOM is created
Now we have:
Previous Virtual DOM
New Virtual DOM

Step 4: Diffing
React compares both trees.

Step 5: Reconciliation
React updates ONLY the changed parts in the real DOM.

What Is Virtual DOM?

Virtual DOM is:

A lightweight JavaScript object

Representation of the real DOM

Stored in memory

React never directly manipulates DOM immediately.
It calculates changes first.
