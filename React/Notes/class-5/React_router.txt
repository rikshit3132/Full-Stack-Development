===== What is React Router?====
-->> React Router is a library that allows 
navigation between components without reloading 
the page.

-->> In normal websites:
-->> When you click a link â†’ page reloads
In React:
-->> We want Single Page Application (SPA)
-->> Only the component changes
-->> Page does NOT reload


=======What is <BrowserRouter>?===

-->> Think of BrowserRouter as the manager of routing
system.
It:

-->> Watches URL changes
-->> Decides which component to show
-->> Uses browser history API
->> Where to put it?
-->> Usually in main.jsx or index.js
Example:

import { BrowserRouter } from "react-router-dom";
ReactDOM.createRoot(document.getElementById("root")).render(
  <BrowserRouter>
    <App />
  </BrowserRouter>
);

 -->> Wrap your entire app with it
 -->> You use it ONLY ONCE

========What is <Routes> and <Route>?========
1) <Routes>
-->> It is a container.
-->> It looks at current URL and checks which 
->> Route matches.

2) <Route>

It defines:
-->> path
-->> which component to show
Example:
<Routes>
  <Route path="/" element={<Home />} />
  <Route path="/about" element={<About />} />
  <Route path="/contact" element={<Contact />} />
</Routes>

======== What is <Link>? ======

-->> Link is used for navigation.

->> Wrong way (reloads page):
<a href="/about">About</a>

-->> Correct way:
<Link to="/about">About</Link>
-->> It changes URL without refreshing page.
