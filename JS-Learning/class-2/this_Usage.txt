### this

-> this is a special keyword that refers 
to an object — but which object it refers to 
depends on how the function is called.

 ### Simple Definition
 this = the object that is calling the function

 Different Cases of this:-

1. Global Scope
console.log(this);
In browser → this refers to window object
In Node.js → this refers to empty object {} (module scope)

2. Inside a Regular Function
function show() {
  console.log(this);
}
show();
In browser → window
In strict mode → undefined

3.  Inside an Object Method
const user = {
  name: "Rikshit",
  greet: function() {
    console.log(this.name);
  }
};
user.greet();
Output: "Rikshit"
Here, this = user object

4.  Arrow Functions
const user = {
  name: "Rikshit",
  greet: () => {
    console.log(this.name);
  }
};

user.greet();
Output: undefined

Arrow functions do NOT have their own this
They take this from their surrounding (lexical) scope

5.  Inside Constructor Function
function Person(name) {
  this.name = name;
}

const p1 = new Person("Rikshit");
console.log(p1.name);

this refers to the new object being created
