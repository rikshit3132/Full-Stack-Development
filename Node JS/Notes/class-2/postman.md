# What is Postman?
- Postman is a tool to test APIs
- You can:
  - Send GET, POST, PUT, PATCH, DELETE
  - Send JSON data
  - See response easily
# Why Postman is used?
- Because:
  - Browser is limited 
  - Postman gives full control 
- Used by:
 - Backend developers
 - Testers
 - Companies
  
# How to Use Postman 

- Open Postman
- Select method (GET / POST / etc)
- Enter URL:
  - http://localhost:3000/notes
- For POST request
  - Go to Body → raw → JSON
- Add:
```json
{
  "title": "DSA",
  "content": "Practice daily"
}
```
- Click Send
