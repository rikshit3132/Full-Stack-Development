### Why use Node JS?
1. Same Language Everywhere
 - With Node.js:
   - Frontend → JavaScript
   - Backend → JavaScript
 - No need to learn multiple languages like:
  - Java (backend)
  - JS (frontend)

2. Very Fast (Non-blocking nature)
- Node.js uses:
  - Google Chrome V8 engine
  - Event-driven, non-blocking system
- Meaning:
 - Handles many users at once
 - Doesn’t get stuck waiting
- Example:
 - 1000 users request data → Node handles efficiently
  
3.Best for Real-Time Applications
 - Node.js is perfect for:
  - Chat apps (like WhatsApp)
  - Live notifications
  - Online games
  - Streaming apps
- Because it can send/receive data instantly
  
4. Huge Ecosystem (npm)
 - npm = package manager of Node.js
 - Millions of libraries available
- You don’t build everything from scratch:
  - Auth → ready package
  - API → ready tools
  - Validation → ready

5.Great for APIs & Backend Services
- Node.js is widely used for:
 - REST APIs
 - Microservices
 - Backend for web/mobile apps
- Example flow:
  - React frontend → Node backend → Database
